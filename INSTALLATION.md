# AppForge by OYARO - Installation Guide

## 📥 Download

### Method 1: GitHub Releases (Recommended)

1. Go to [Releases Page](https://github.com/emmanueloyaro123-jpg/appforge-oyaro/releases)
2. Download latest `AppForge-by-OYARO.apk`
3. Save to your phone's Downloads folder

---

## ⚙️ Prerequisites

- **Android Version:** 11.0 or higher
- **RAM:** 2GB minimum (4GB+ recommended)
- **Storage:** 500MB free space
- **Network:** WiFi or mobile data (optional, for AI)

---

## 🔓 Enable Unknown Sources

Android blocks installation of apps from unknown sources by default.

### Step 1: Open Settings
1. Open **Settings** app
2. Tap **Security** or **Privacy**
3. Look for "Unknown sources" or "Install unknown apps"

### Step 2: Enable Installation
1. Tap on your file manager/browser that will download APK
2. Enable "Allow from this source"
3. Or toggle "Unknown sources" globally

### Different Android Versions

**Android 11-12:**
- Settings → Security → Unknown sources
- Or Settings → Apps → Special app access → Install unknown apps

**Android 13+:**
- Settings → Apps → Special app access → Install unknown apps
- Select the app (browser/file manager) you'll use to install from

---

## 📱 Installation Steps

### Step 1: Locate APK
1. Open **Files** app
2. Navigate to **Downloads**
3. Find **AppForge-by-OYARO.apk**

### Step 2: Install
1. Tap the APK file
2. Android will show installation confirmation
3. Tap **Install**
4. Wait for installation (usually 1-2 minutes)

### Step 3: Verify Installation
1. You'll see "App installed" message
2. Or go to Settings → Apps → "AppForge"
3. The app is now on your device

---

## 🚀 Launch AppForge

### First Launch
1. Open **App Launcher** (home screen)
2. Scroll to find "AppForge" or search for it
3. Tap the AppForge icon
4. App will start (may take 10-15 seconds first time)

### Grant Permissions
AppForge may request permissions:
- Storage access
- Camera (optional)
- Contacts (optional)
- Location (optional)

Grant as needed.

## ⚠️ Troubleshooting

### "App not installed"
**Solutions:**
1. Delete the APK and re-download
2. Ensure device has 500MB+ free storage
3. Restart your phone
4. Try a different file manager
5. Download from a different browser

### "Install from Unknown Sources" not working
**Solutions:**
1. Go to Settings → Privacy → Permission manager
2. Ensure Files app has necessary permissions
3. Try using a different browser to download
4. Check if device is in restricted mode

### Installation stuck or slow
**Solutions:**
1. Close other apps
2. Free up RAM
3. Check WiFi connection
4. Try over cellular data
5. Restart device

### App crashes on first launch
**Solutions:**
1. Uninstall: Settings → Apps → AppForge → Uninstall
2. Re-download and reinstall
3. Grant all requested permissions
4. Ensure device has 2GB+ RAM free
5. Update Android OS
