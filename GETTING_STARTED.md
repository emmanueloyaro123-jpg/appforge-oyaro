# Getting Started with AppForge by OYARO

## 📱 Installation

### Step 1: Download
1. Go to [Releases](https://github.com/emmanueloyaro123-jpg/appforge-oyaro/releases)
2. Download `AppForge-by-OYARO.apk`
3. Save to your phone's Downloads folder

### Step 2: Enable Installation
1. Open Settings → Security
2. Toggle "Install from Unknown Sources"
3. Select your browser to allow

### Step 3: Install
1. Open Files app
2. Navigate to Downloads
3. Tap `AppForge-by-OYARO.apk`
4. Tap "Install"
5. Wait for completion

### Step 4: Launch
1. Find "AppForge" in app launcher
2. Tap to open
3. Grant permissions when prompted
4. You're ready!
